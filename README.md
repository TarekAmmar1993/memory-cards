## Prerequisites

- _Node.js_ v22
- [_pnpm_](https://pnpm.io/installation) v9

## Quickstart

1. Clone the project
2. Run `pnpm i`
3. Run `pnpm dev`
4. Open your browser and visit [http://localhost:3000](http://localhost:3000)
5. To check the typecheck, run `pnpm typecheck`
